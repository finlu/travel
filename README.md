解决移动端点击300毫秒延时问题:

安装fastclick的js库：

1. `npm install fastclick --save`

2. 在`main.js`中使用`flat click`

   ```js
   import fastClick from 'fastclick'
   fastClick.attach(document.body)
   ```


安装`stylus`

```shell
npm install stylus —save
npm install stylus-loader —save
```

给文件起别名：

在webpack中仿照 `@` 代表`src` 目录的方式，给指定的文件夹起别名，之后在import语句中就可以使用别名进行引入。

在`css`中`import` 变量的时候，需要在路径前加上`~`

清除抖动效果加宽高自适应:

```stylus
.wrapper
  overflow hidden
  width 100%
  height 0
  padding-bottom 31.25%  宽高比
```

或者:

```sty
.wrapper
  width 100%
  height 31.25vm.  有兼容性问题
```

显示`…`的效果：

```stylus
overflow hidden
white-space nowrap
text-overflow ellipsis
```

stylus中封装方法`mixin`：

```styl
ellipsis()
  overflow hidden
  white-space nowrap
  text-overflow ellipsis
```

在stylus中使用：

```sty
ellipsis()
```
